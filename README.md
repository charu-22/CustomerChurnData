#  Customer Churn Prediction

This project predicts whether a customer will churn based on telecom data.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature encoding
- Classification model (Logistic Regression)

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn

## Goal
Identify customers likely to churn and understand key influencing factors.
